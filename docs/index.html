Lett å blande

כָּֽתְבָה Qal Perfektum 3fs כָּתַב,oversatt «hun skrev» כָּתְבָה Qal Imperativ 2ms כָּתַב (medָ ה endingen), oversatt «(du) Skriv»
כָּֽתְבוּ Qal Perfektum 3cp כָּתַב, oversatt «de skrev» כִּתְבוּ Qal Imperativ 2mp כָּתַב, oversatt «(dere) Skriv!»
בְּנִי Qal Imperativ 2fs בָּנָה, oversatt «(du) Skriv!» בְּנִי Maskulinum singularis בֵּן, med 1cs pronomenial suffix, ovseratt «min sønn»
בָּנוּ Qal Perfektum 3cp בָנָה, «de bygde» בָּ֫נוּ Perposisjon בְּ, med 1cp pronomenial suffix, «mot oss» בְּנוּ Qal Imperativ 2mp בָּנָה, «(dere) Bygg!»
עָלוּ Qal Perfektum 3cp עָלָה, «de gikk opp» עֲלוּ Qal Imperativ 2mp עָלָה, «(dere) Gå opp!»
